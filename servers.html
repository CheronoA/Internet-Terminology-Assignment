🖥️ Web Servers

A web server is hardware or software that stores, processes, and delivers web pages to clients (browsers) over HTTP/HTTPS. It listens on particular ports for incoming requests and responds with files or dynamically generated content.

🌍 Popular Web Servers

🐘 Apache
⚡ Nginx
🪟 IIS
☕ Tomcat
🚀 Node.js

🔄 How a Web Server Works

  1. Listen - Waits for HTTP requests on port 80/443
  2. Receive - Browser sends a request
  3. Process - Maps URL to file or executes code
  4. Respond - Sends HTTP response with status code
  5. Log - Records the request
