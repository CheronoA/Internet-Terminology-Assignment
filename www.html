🌐 The World Wide Web

What is the World Wide Web?

The World Wide Web (often just called "the web") is like a giant digital library. You can sit at your computer or phone, open a browser, and visit millions of connected pages. These pages have text, images, videos, links — almost anything you can imagine. The web runs on top of the internet, which is the physical network of cables and routers.

✏️ My own way of remembering it: The internet is like the roads, highways, and postal system. The web is like all the shops, houses, and libraries you can visit using those roads. You need roads to get there, but the roads aren't the same thing as the buildings.

🎓 Who invented it and why?

A British scientist named Tim Berners-Lee invented the web in 1989 while working at CERN (a physics lab in Switzerland). He wanted a way for scientists around the world to easily share documents and research papers. His first proposal was called "Information Management" — and it slowly grew into the web we know today.

The first website went live in 1991. It was just a simple page explaining what the web was and how to set up a server. You can still visit it today: info.cern.ch.

🧩 How the web works (simple version)

  • You type a URL — like www.google.com — into your browser.
  • DNS lookup — a directory service translates that name into a number (IP address).
  • Browser sends an HTTP request — asking a server for the page.
  • Server responds — sends back HTML, CSS, images, etc.
  • Browser renders the page — you see it on your screen.

All of this usually happens in less than a second. It's honestly kind of magic when you think about it.

📖 Web vs Internet — what's the difference?

This confused me at first. Here's how I understand it:

  • Internet = hardware + protocols (cables, routers, IP addresses, TCP/IP). It connects computers together.
  • World Wide Web = one of the services that runs on the internet (like email or FTP). It's specifically about web pages, links, URLs, and HTTP.

The internet existed before the web. You can have the internet without the web (e.g., sending an email). But you cannot have the web without the internet.

  • Doing research for assignments on Wikipedia 📚
  • Online shopping (Amazon, Jumia) 🛍️
  • Using Google Maps to find places 🗺️

🔗 Why the web matters

The web changed everything. Before it, information was hard to share and update. Newspapers, TV, and books were the main sources of news. Now anyone with a connection can publish a blog, start an online store, or learn almost anything for free. The web made knowledge democratic — available to everyone, not just experts.

But it also has problems (fake news, privacy issues, centralization). That's why newer versions like Web 3.0 and Web 4.0 are trying to fix some of those flaws.

💭 What I think: The web is one of the most important inventions in human history — right up there with the printing press. It connects people, ideas, and cultures in a way that was impossible 40 years ago.
