📡 HTTP (Hypertext Transfer Protocol)

HTTP is the application-layer protocol used to transfer data on the World Wide Web. It defines how requests are formatted by clients (browsers) and how servers respond.

🔑 Key HTTP Methods

GET - Retrieve data from the server
POST - Submit data to be processed
PUT - Update existing resource
DELETE - Remove a resource

📊 HTTP Status Codes

200 OK 301 Redirect 404 Not Found 500 Server Error

🔄 HTTP Request-Response Cycle

  1. User enters URL - Types or clicks a link
  2. DNS resolution - Translates domain to IP
  3. TCP connection - 3-way handshake
  4. HTTP request sent - Browser asks for resource
  5. Server processes request - Executes logic
  6. HTTP response returned - Status code + content
  7. Browser renders page - Parses and displays
  8. Connection closed/kept alive
