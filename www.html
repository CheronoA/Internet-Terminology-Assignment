🌐 World Wide Web (WWW)

The World Wide Web (WWW) is a system of interlinked hypertext documents and multimedia content accessed through the internet using web browsers. The web is not the internet itself — it is a service that runs on top of the internet infrastructure.

💡 Key Distinction: The Internet is the physical network (cables, routers, protocols), while the Web is the information system (websites, pages, links) that runs on top of it.

📅 History of the Web

1989
Tim Berners-Lee proposes the Web at CERN
1990
First web browser and server created
1991
First website goes live
1993
CERN releases Web to public domain
1995
Internet Explorer launched

🔧 How the Web Works

  1. User enters URL - Type a web address or click a link
  2. Browser sends HTTP request - Asks server for the webpage
  3. Server processes request - Locates the file or generates content
  4. Server sends HTTP response - Returns HTML, CSS, JS, images
  5. Browser renders page - Parses code and displays visually

📋 Core Web Technologies

  • HTML - Structure and content of web pages
  • CSS - Styling and visual presentation
  • JavaScript - Interactivity and dynamic behavior
  • HTTP/HTTPS - Protocol for transferring data
  • URL - Address system for locating resources

🌍 Web vs Internet

  • Internet - Physical infrastructure (cables, routers, IP addresses)
  • World Wide Web - Information system (websites, hyperlinks, URLs)
  • Other internet services: Email (SMTP), File Transfer (FTP), VoIP (Skype)
