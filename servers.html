Web Servers

A web server is a computer system that stores, processes, and delivers web pages to clients (browsers) over the Internet.

Key facts about web servers:

  • Responds to HTTP/HTTPS requests from browsers
  • Examples: Apache, Nginx, Microsoft IIS
  • Can host multiple websites on one server
  • Each website needs a domain name to be accessible
