URL (Uniform Resource Locator)

A URL (Uniform Resource Locator) is the web address used to access a specific resource on the Internet, like a webpage, image, or file.

Parts of a URL:

  • Protocol (e.g., https://)
  • Domain name (e.g., google.com)
  • Path (e.g., /search)
  • Query parameters (e.g., ?q=hello)

Example: https://www.example.com/page?id=123
