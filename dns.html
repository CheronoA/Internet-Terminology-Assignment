📞 DNS (Domain Name System)

DNS is the internet's phonebook. It translates human-readable domain names (like google.com) into machine-readable IP addresses (like 142.250.190.46) so computers can locate each other.

🔍 How DNS Works - Step by Step

Step 1: User types "www.example.com" into browser
Step 2: Browser checks local cache (has this site been visited before?)
Step 3: If not in cache, query goes to Recursive DNS Resolver (usually your ISP)
Step 4: Resolver queries Root DNS Server → finds .com TLD server
Step 5: TLD server points to Authoritative DNS Server for example.com
Step 6: Authoritative server returns the IP address
Step 7: Browser receives IP and connects to the website

🗂️ DNS Hierarchy

Root (.)
├── .com (Commercial)
│ └── google.com
│ └── www.google.com
├── .org (Organizations)
│ └── wikipedia.org
├── .net (Network)
├── .edu (Education)
└── Country Codes (.ke, .uk, .us, .jp)

📋 Types of DNS Records

  • A Record - Maps domain to IPv4 address
  • AAAA Record - Maps domain to IPv6 address
  • CNAME Record - Alias of one domain to another
  • MX Record - Mail exchange servers for email
  • TXT Record - Text information (verification, security)
