🌐 World Wide Web — The System of Linked Pages

The World Wide Web (WWW) is a system of interlinked hypertext documents and multimedia content that you access through the internet using a web browser. The web is not the same thing as the internet — it's a service that runs on top of the internet infrastructure.

✏️ My simple way to think about it: The internet is like the roads, highways, and postal system. The web is like all the shops, houses, and libraries you can visit using those roads. You need the internet to access the web, but the web is just one of many things you can do on the internet (email, FTP, gaming are others).

📅 History of the Web — Who Invented It?

A British scientist named Tim Berners-Lee invented the web in 1989 while working at CERN (a physics lab in Switzerland). He wanted a way for scientists around the world to easily share documents and research papers. The first website went live in 1991 — it was just a simple page explaining what the web was and how to set up a server.

📌 1989: Tim Berners-Lee proposes the Web 📌 1990: First web browser and server created 📌 1991: First website goes live 📌 1993: CERN releases the Web to the public (free!) 📌 1995: Internet Explorer launched 📌 2004: Web 2.0 era begins

🧩 How the Web Works — Simple Version

  1. You type a URL — like www.google.com — into your browser
  2. DNS lookup — a directory service translates that name into an IP address
  3. Browser sends an HTTP request — asking a server for the page
  4. Server responds — sends back HTML, CSS, images, etc.
  5. Browser renders the page — you see it on your screen

All of this usually happens in less than a second. It's honestly kind of amazing when you think about it.

📖 Web vs Internet — What's the Difference?

  • Internet = hardware + protocols (cables, routers, IP addresses, TCP/IP). It connects computers together.
  • World Wide Web = one of the services that runs on the internet (like email or FTP). It's specifically about web pages, links, URLs, and HTTP.

The internet existed before the web (since the 1960s). You can have the internet without the web (like sending an email). But you cannot have the web without the internet.

🔧 Core Technologies That Make the Web Work

  • HTML — gives structure and content to pages
  • CSS — adds style and visual design
  • JavaScript — makes pages interactive (buttons, animations, forms)
  • HTTP/HTTPS — the protocol for transferring data
  • URLs — the addressing system for finding resources
