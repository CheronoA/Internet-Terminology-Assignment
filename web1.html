📖 Web 1.0 (1991-2004) - The Read-Only Web

Web 1.0 was the first generation of the World Wide Web, often called the "Read-Only Web." Users could only consume content — there was little to no interaction or user-generated content.

🔑 Key Characteristics of Web 1.0

  • Static pages - Content was fixed and rarely updated
  • Read-only - Users could only view information
  • No user interaction - No comments, likes, or reviews
  • Information-centric - Websites were like digital brochures
  • Owners controlled content - Only webmasters could post

📋 Examples of Web 1.0 Sites

  • Yahoo! (early directory style)
  • GeoCities (personal static pages)
  • Britannica Online (encyclopedia)
  • Early news websites (CNN, BBC)
  • Basic company brochure sites

⚙️ Technologies Used

  • HTML (basic, no CSS3 or HTML5)
  • HTTP (basic protocol)
  • URLs (addressing)
  • GIF buttons and animated banners
  • Guestbooks (simple feedback forms)
💡 Fun Fact: The first website ever created is still online at info.cern.ch. It was published on August 6, 1991.
