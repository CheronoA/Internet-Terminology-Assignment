📁 FTP (File Transfer Protocol)

FTP (File Transfer Protocol) is a standard network protocol used to transfer files between a client and a server over a TCP/IP network. It uses separate command and data channels.

🔧 How FTP Works

FTP Client ── Port 21 (Commands) ──→ FTP Server
FTP Client ←── Port 20 (Data) ──→ FTP Server

📋 Common FTP Clients

  • FileZilla (most popular)
  • WinSCP
  • Cyberduck
  • Command-line FTP

🔒 FTP vs FTPS vs SFTP

  • FTP - No encryption (passwords sent in plain text)
  • FTPS - FTP with SSL/TLS encryption
  • SFTP - SSH File Transfer Protocol (secure)

📋 Common FTP Commands

  • USER - Send username
  • PASS - Send password
  • LIST - List files in directory
  • GET - Download file from server
  • PUT - Upload file to server
  • QUIT - End session
