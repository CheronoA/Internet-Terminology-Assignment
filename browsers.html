🧭 Web Browsers

A web browser is client software that retrieves, processes, and renders web content. It sends HTTP requests to web servers, receives HTML/CSS/JS responses, and displays them as visual pages.

🌍 Popular Web Browsers

🌐 Google Chrome
🦊 Mozilla Firefox
🧭 Safari
🔷 Microsoft Edge
📱 Opera

⚙️ How Browsers Work

  1. Navigation - User enters URL or clicks a link
  2. DNS Lookup - Browser finds server IP address
  3. HTTP Request - Browser requests the webpage
  4. Parse HTML - Browser builds DOM tree
  5. Render - Display the page to user
