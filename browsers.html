🧭 Web Browsers

A web browser is the software you use to visit websites. When you type a URL or click a link, the browser fetches the webpage from a server and displays it on your screen.

✏️ My way to think about it: A browser is like a restaurant waiter. You tell them what you want, they go to the kitchen (server), bring back your food (webpage), and serve it to you.

🌍 Popular browsers

  • Google Chrome — popular but uses lots of RAM
  • Mozilla Firefox — more privacy-focused
  • Microsoft Edge — new version is actually good
  • Safari — only on Apple devices
  • Opera — has built-in VPN
  • Brave — blocks ads and trackers

⚙️ How a browser works

  1. You enter a URL or click a link
  2. Browser finds the server's IP address (DNS lookup)
  3. Browser sends an HTTP request to the server
  4. Server sends back HTML, CSS, JavaScript, images
  5. Browser reads the code and displays the page
