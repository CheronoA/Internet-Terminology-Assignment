DNS (Domain Name System)

The Domain Name System (DNS) is like the phonebook of the Internet. It translates human-friendly domain names (like google.com) into IP addresses (like 142.250.190.46) that computers use to identify each other.

Key facts about DNS:

  • DNS servers convert domain names to IP addresses
  • Without DNS, we would need to memorize numbers like 192.168.1.1
  • DNS lookup happens in milliseconds
  • There are 13 root DNS server types worldwide
