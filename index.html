📘 My Internet Terminology Notebook

This website is my personal glossary for the Web Application Development assignment. I've tried to explain each term in my own words, using examples and diagrams that make sense to me. I hope you enjoy reading it as much as I enjoyed making it.

✏️ A note from me: The internet isn't just something we use every day — it's a whole world of protocols, servers, browsers, and ideas working together. Below are the key terms I studied. Click any card to learn more.

📚 Topics I've Covered

🌐 World Wide Web
The system of linked pages we browse every day.
🔌 TCP/IP
The fundamental language of the internet.
🧭 Web Browsers
Our window to the web (Chrome, Firefox, etc.)
🖥️ Web Servers
The computers that store and deliver websites.
🔗 URL
The address bar — how we find things online.
📞 DNS
The phonebook of the internet.
📡 HTTP
The protocol browsers and servers use to talk.
🏢 Intranet
A private network inside a company.
🤝 Extranet
A shared space between companies.
📊 Multitier Architecture
How big apps are organized into layers.
📁 FTP
The old way to transfer files online.
📝 HTML
The skeleton of every webpage.
📖 Web 1.0
The early read-only web.
✍️ Web 2.0
The interactive social web.
🧠 Web 3.0
The decentralized, AI-powered web.
🚀 Web 4.0
The future — symbiotic web with IoT and AI.
