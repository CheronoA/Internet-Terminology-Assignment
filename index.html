📚 Internet Terminology Glossary

Your complete guide to understanding the fundamental concepts of the World Wide Web

🌐

World Wide Web

A system of interlinked hypertext documents accessed via the internet using web browsers.
🔌

TCP/IP

The fundamental communication protocol suite governing how data is transmitted across the internet.
🧭

Web Browsers

Client software that retrieves, processes, and renders web content.
🖥️

Web Servers

Hardware or software that stores, processes, and delivers web pages.
🔗

URL

A reference (address) to a specific resource on the web.
📞

DNS

The internet phonebook that translates domain names to IP addresses.
📡

HTTP

The application-layer protocol used to transfer data on the web.
🏢

Intranet

A private network using internet technologies accessible only to organization members.
🤝

Extranet

An extension of an intranet that allows authorized external access.
📊

Multitier Architecture

Separation of web applications into distinct layers: presentation, logic, and data.
📁

FTP

Standard network protocol used to transfer files between client and server.
📝

HTML

The standard markup language used to create and structure web pages.
📖

Web 1.0

The read-only web (1991-2004) - static pages, no user interaction.
✍️

Web 2.0

The read-write web (2004-2016) - social media, user-generated content.
🧠

Web 3.0

The semantic, decentralized web using blockchain and AI.
🚀

Web 4.0

The symbiotic web - AI-driven, IoT integrated, intelligent agents.
