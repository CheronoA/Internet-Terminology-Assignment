🔗 URL (Uniform Resource Locator)

A URL is a reference (address) to a specific resource on the web. It specifies the protocol, host, path, and optionally query parameters or fragments.

https://www.example.com:443/products/item123?id=42&color=red#reviews
1. Scheme (https://)
How to access the resource
2. Subdomain (www)
Service section within domain
3. Domain Name (example.com)
Human-readable host address
4. Port (:443)
Network endpoint (default: 80, 443)
5. Path (/products/item123)
Location of resource on server
6. Query String (?id=42&color=red)
Key-value parameters for dynamic pages
7. Fragment (#reviews)
Points to section within page
