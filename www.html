🌐 World Wide Web (WWW)

The World Wide Web (WWW) is a system of interlinked hypertext documents and multimedia content accessed through the internet using web browsers.

💡 Key Distinction: The Internet is the physical network, while the Web is the information system that runs on top of it.

📅 History of the Web

1989
Tim Berners-Lee proposes the Web
1990
First browser and server created
1991
First website goes live
1993
CERN releases Web to public

📋 Core Web Technologies

  • CSS - Styling and presentation
  • JavaScript - Interactivity
  • HTTP/HTTPS - Data transfer protocol
